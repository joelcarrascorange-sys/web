01 — VISIÓN GENERAL

El Sistema Albegestio

Punto de venta autónomo que dispensa coleccionables mediante pago electrónico. Integra componentes de última generación para una experiencia fluida y segura.

⬛

Interfaz táctil 7"

Modo kiosko con animaciones fluidas y guía visual paso a paso para el usuario.

◈

Ingenico Self 5000

Terminal NFC/EMV con certificación PCI-DSS delegada. Pago seguro sin acceso de la Pi a datos bancarios.

⟳

Husillo Archimedes 3D

Mecanismo de dispensación de precisión con sensor IR de confirmación y reintentos automáticos.

◉

Raspberry Pi 4B

Unidad de control ejecutando Node.js modular. Mínimo 2GB RAM, recomendado 4GB.

05 — FLUJO DE VENTA

Timeline completo

T+0.0s
Usuario presiona "Obtener" en pantalla táctil
T+0.5s
Raspberry Pi activa pago en terminal Self 5000
T+Xs
Banco autoriza transacción (código "00")
T+X.1s
Servo activa el husillo dispensador
T+X.5s
Sensor IR confirma artículo dispensado ✓
06 — RECUPERACIÓN

Protocolos de error

TIMEOUT

Pago sin respuesta → Revert a IDLE tras 30s

WEBSOCKET

Pérdida de conexión → Reconexión automática en 3s

DISPENSACIÓN

Fallo de mecanismo → Log + alerta + reembolso

SIN SENSOR IR

2 reintentos automáticos → Bloqueo de seguridad
